# 💸 Калькулятор чаевых

Простой и красивый калькулятор для расчета чаевых в ресторане.

## 🚀 Демо
[Посмотреть калькулятор](https://opakinalarisa7-netizen.github.io/tip-calculator)

## 📱 Функции
- Расчет чаевых в процентах (0-30%)
- Разделение счета на любое количество человек
- Мгновенный пересчет при изменении данных
- Красивая адаптивная верстка

## 🛠 Технологии
- HTML5
- CSS3 (Flexbox, анимации, градиенты)
- JavaScript (чистый JS)
